The programmes main file is Visper.py which contains the options to be done.

However, the main calculations are performed within the classes. The main class is Spect where all general informations needed are computed and stored. For the different models, the classes FC-spect (as well as its child classHR), CFC, URDR and DR. are derived from Spect.

Moreover, there are helper classes for reading the quantum chemical log files, alignment of molecules,output etc.

The pycco-formatted source files in total are the following:

  • main program code
  • Class Spect
  • Derived Classes FC/CFC
  • Derived Classe HR
  • Derived Classes DR/URDR
  • Dependent Class Btree
  • Class Read
  • Class MultiPart
  • atoms_align
  • file_handler
  • normal_modes
  • output_spect
