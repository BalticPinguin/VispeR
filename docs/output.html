General information

The output is written in a file specified by out: in the input file. Most of the information (except the final information asked by the options in the input-file) is printed only if the logging-level is low enough. For details about the possible levels see print=....

If a file with the name of the log-file exists already, the information will be appended to it.

Each output starts by a header .

The output of every run starts with the string calculations to be done: followed by a number which encodes the options set. Here 1 denotes the calculation of Huang-Rhys-factors (and Duschinsky-matrix), 2 denotes the calculation of a line-spectrum in FC-scheme, 4 is the calculation of a line-spectrum with Duschinsky-rotation and 8 means that a broadened spectrum has to be calculated. Number printed is the sum of these numbers and determines uniquely, what has to be done.

Further, each of these tasks has it's own logging-flag (how detailed the print-out is).

Output from the calculation of Huang-Rhys factors

This part will be introduced by the line Use method ... for the calculation of all quantities. in any case.

spectrum in Franck-Condon picture

The calculation of FC-spectra is introduced by Calculate the line-spectrum in FC-picture with %s excitations. in the log-file (besides some minor information as temperature of the system, depending on the niveau of output).

The line-spectrum in onp-particle-approximation (independent of the option to nPA) is printed if loglevel=0. It starts with the string Line-spectrum in One-Particle approximation: and has frequencies (in cm^-1) in first, intensities in second column and the mode that changes it's state in third culumn. With these informations, a full nPA-spectrum can be calculated.

spectrum including Duschinsky-effect

The calculation of spectra taking Duschinsky-effect into account starts with Use the model ... for the calculation of linespectrum. followed by additional statements depending on the model used.

if logging=0, the change of Cartesian coordinates is printed after the key words changes of Cartesian coordinates .

if logging is 0 or 1, the Duschinsky-rotation matrix and displacement vector are printed after the key words Duschinsky rotation matrix, state and Duschinsky displacement vector: respectively.
