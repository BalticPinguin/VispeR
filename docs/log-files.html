Introduction

VispeR understands different input-formats and tries to calculate the requested data with that values that are given.

However, some data are needed and not all possible print-formats can be supported.

Gaussian-Files

option Freq is needed to calculate the frequencies and print the Hessian respectively.

Gaussian has the feature that it turns the molecules always to its moment of intertia axis. This, however, has disadvantages here: First, the direction of an axis is not defined by it and second for symmetric molecules the order of moments of inertia changes due to (almost) degeneracy. One should therefor use the NoSymm-option to avoid axis-flips.

Also, it is (especially for larger molecules) recommended to use the formcheck-files (.fchk or .FCkh) as input since the data therein are numerically more stable.

NWChem-Files

To get frequency information, specify task dft freq or the respective method of choise.

GAMESS-Files

Gamess-files used to be supported and maybe will be later on, but at the moment calculations trying to use them will fail and through errors.
